# DPDA Inbox Quick Reference Guide

## For DPDA Users

### Accessing Your Inbox

1. Click "DPDA Inbox" in the left sidebar
2. You'll see all forwarded files from P1-P10 waiting for your review

### Understanding the Dashboard

**Status Summary Cards** (Top of page):
- **Total**: All forwarded files
- **Pending**: Files waiting for your decision
- **Approved**: Files you've approved
- **Disapproved**: Files you've rejected
- **Returned**: Files sent back to senders

### Searching & Filtering

**Quick Search** (Top search bar):
- Type file title, sender name, or any notes
- Results update instantly

**Advanced Filters** (Click "Filters" button):
- **Status**: Show only files with specific status
- **Sender**: Show files from specific P1-P10
- **Priority**: Focus on high/urgent items
- **Sort**: Order by date, priority, or sender

### Reviewing a File

1. **Click any file card** to open detailed view
2. **Review section**:
   - File title and sender info
   - Document type and priority
   - Sender's forwarding notes
   - Main document (click to download)
   - Attached files (click to view)

### Taking Action

#### To Approve:
1. Click "Approve" button
2. Optionally add approval notes
3. Click "Confirm"
4. File moves to "Approved" status

#### To Disapprove:
1. Click "Disapprove" button
2. Enter reason for disapproval
3. Add detailed feedback
4. Click "Confirm"
5. File moves to "Disapproved" status

#### To Add Comments:
1. Click "Add Comment" button
2. Type your comment/feedback
3. Click "Confirm"
4. File moves to "Returned with Comments" status
5. Comment is stored for sender to review

#### To Forward Back:
1. After approving/disapproving, click "Forward Back to Sender"
2. File is sent back with your decision and comments
3. Sender receives automatic notification
4. File status changes to "Returned"

### File Status Meanings

| Status | Meaning | Action |
|--------|---------|--------|
| 🟡 Pending Review | Waiting for your decision | Review and take action |
| 🟢 Approved | You've approved it | Ready to forward back |
| 🔴 Disapproved | You've rejected it | Ready to forward back |
| 💬 Returned with Comments | Has feedback | Ready to forward back |
| ↩️ Returned | Already sent back | View only (archived) |

### Priority Levels

- 🟡 **Low**: Can wait, routine matters
- 🔵 **Medium**: Normal priority (default)
- 🟠 **High**: Important, needs attention soon
- 🔴 **Urgent**: Critical, needs immediate attention

### Tips & Tricks

1. **Bulk Review**: Use filters to group similar files
2. **Quick Notes**: Add comments for future reference
3. **Search History**: Browser remembers your last search
4. **Mobile Friendly**: Use on tablet or phone
5. **Pagination**: 12 files per page, use page buttons to navigate

### Keyboard Shortcuts

- `Ctrl/Cmd + F`: Open search
- `Escape`: Close modal/popup
- `Tab`: Navigate between buttons

### Common Tasks

**I want to review all P1 files:**
1. Click "Filters"
2. Select "Sender: P1 - Records Officer"
3. Browse through results

**I want to focus on urgent items:**
1. Click "Filters"
2. Select "Priority: Urgent"
3. Review and act on these first

**I want to see my recent approvals:**
1. Click "Filters"
2. Select "Status: Approved"
3. Sort by "Newest First"

**I want to find a specific file:**
1. Use the search bar at top
2. Type part of the filename or sender info
3. Results update as you type

### What Happens Next

**After You Approve/Disapprove:**
1. Document status updates immediately
2. Sender receives a notification
3. You can add comments before forwarding
4. File stays in your inbox until forwarded back

**After You Forward Back:**
1. File is returned to sender with your decision
2. File moves to "Returned" status
3. File appears in your history (not pending anymore)
4. Sender can now take next steps

### Getting Help

**If you see an error:**
1. Try refreshing the page
2. Check your internet connection
3. Log out and log back in
4. Contact administrator if problem persists

**If a file won't load:**
1. Check if you have the right permissions
2. Try opening in a different browser
3. Clear browser cache
4. Contact administrator

**If you can't find a file:**
1. Check different status tabs
2. Try removing filters
3. Use search bar with less specific terms
4. Contact the sender

### Notifications

When P1-P10 forwards you a file:
- ✉️ Notification appears in your dashboard
- 📧 Email notification (if enabled)
- 🔔 Browser notification (if enabled)

- Sender gets notification with your decision
- Comments are included in notification
- Sender can view document immediately

### FAQ

**Q: Can I see files from other DPDAs?**
A: No, each DPDA has their own inbox

**Q: Can I undo an approval?**
A: No, but you can forward back with additional comments

**Q: How long do files stay in my inbox?**
A: Until you forward them back to sender

**Q: Can I assign files to other DPDA staff?**
A: No, but you can add comments with instructions

**Q: Can senders see my comments before I forward back?**
A: No, only after you click "Forward Back"

**Q: What if I need more time to review?**
A: Add comments to defer and set expectations
